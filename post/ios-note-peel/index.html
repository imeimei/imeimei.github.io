  1. 需要集成百度语音SDK

注意事项

抽离思路

  • 基础模块抽离出一个FrameWork

  • 笔记也要抽离出一个FrameWork

  • 笔记模块依赖基础模块

  • 一事通框架也依赖基础模块

  • 一事通引入笔记FrameWork

按照这个思路，抽离出来的基础模块的FrameWork应该要弄成动态链接库的形式。

笔记需要基础模块提供的数据

  • 当前服务器时间
  • InUseId： 当前用户实际使用人的一事通ID
  • deviceId: 当前设备ID
  • platform: 当前设备平台

可能面临的问题

在基础模块中引入的第三方类，如果另外一个第三方工程使用基础模块的FrameWork也用到了这个第三方库，那么这个第三方工程就要把这个第三方库删除以避免冲突。

小技巧

  • 一进入到手机就获取服务器时间和手机的时间，因为手机时间可能不准确，则过段时间后服务器时间计算为: $$ time = 刚进入APP时服务器时间+手机当前时间-刚进入APP时手机时间$$

alert.show

修复alert.show报错时，首先修改Extension中UIAlertController下的show()函数，然后引入 extension UIWindow和extension UIScreen，则可消除错误。

extension UIAlertController {
    func show() {
        var presentViewController = UIApplication.shared.keyWindow?.safeRootViewController
        while(presentViewController?.presentedViewController != nil) {
            presentViewController = presentViewController?.presentedViewController
        }
        
        var topViewController = presentViewController
        if topViewController is UITabBarController {
            topViewController = (topViewController as! UITabBarController).selectedViewController
        }
        if topViewController is UINavigationController {
            topViewController = (topViewController as! UINavigationController).topViewController
        }
        topViewController?.present(self, animated: true, completion: nil)
    }
}

Date().defaultString

在Extension中添加

extension Date {
    var defaultString:String {
        get {
            let formatter = DateFormatter()
            formatter.dateFormat = "yyyy-MM-dd HH:mm:ss"
            return formatter.string(from: self)
        }
    }
    var fileString:String {
        get {
            let formatter = DateFormatter()
            formatter.dateFormat = "yyyyMMddHHmmssSSS"
            return formatter.string(from: self)
        }
    }
    var noteString:String {
        get {
            let formatter = DateFormatter()
            formatter.dateFormat = "yyyy-MM-dd  HH:mm"
            return formatter.string(from: self)
        }
    }
}
